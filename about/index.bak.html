关于考研

[1] 设定目标

目标分数 : 380
政治/70 + 英语/70 + 数学/120 + 专业/120

[2] 复习准备

视频课 + 辅导书 + 笔记 + 真题

  • 基础预习阶段 01月 - 06月
  • 强化学习阶段 07月 - 08月
  • 冲刺提高阶段 09月 - 11月
  • 预测点睛阶段 12月

[3] 复习规划

3-12月, 一共10个月左右

  • 学习阶段 -> 4个月左右
  • 刷题阶段 -> 2个月左右
  • 总结阶段 -> 剩余时间

警告 : 10月31日前完成真题模拟

真题模拟要做到 :

  • 模拟时间
  • 模拟科目顺序
  • 模拟试卷
  • 模拟答题卡

三次模拟成绩最高成绩超过380，保持节奏，分析低于380的情况，继续坚持考试。
三次模拟成绩最高成绩距离380小于20，分析差距，细微调整复试节奏，有重点提高分数。
三次模拟成绩最高成绩超过380大于30，考虑是否换校或要付出更多时间，精力提高分数。

政治: 目标70分

[1] 计划分数 -> 70
70分 = 选择题35分 + 分析题35分
考试180分钟, 按试卷顺序做题即可 :

  • 单选题24分钟, 约1.5分钟/题
  • 多选题42分钟, 约2.5分钟/题
  • 分析题100分钟, 约20分钟/题
  • 剩余14分钟可以查漏补缺

❤ 要求：选择题最多错10个左右；分析题不能跑题

[2] 分值组成
马原 -> 马克思主义基本原理
毛中特 -> 毛泽东思想和中国特色社会主义理论体系概论
史纲 -> 中国近现代史纲要
思修 -> 思想道德修养与法律基础
时政 -> 形势与政策以及当代世界经济与政治

试卷结构:

  • 单项选择题
  • 多项选择题
  • 分析题
    试卷题型结构: 共计38道题目, 满分100分。

单选题：16小题，每小题1分，共16分
多选题：17小题，每小题2分，共34分
分析题：5小题，每小题10分，共50分

马克思主义基本原理约24分
毛泽东思想和中国特色社会主义理论体系概论约30分
中国近现代史纲要约14分
思想道德修养与法律基础约16分
形势与政策以及当代世界经济与政治约16分

⚠️ 复习不要看大学教材 ! 不要看大学教材 ! 不要看大学教材 !

[3] 复习规划

✅ 第一轮：6-7月，1小时/天
划重点：看视频，理解知识点
不需要背，是理解，理解的程度达到开卷考试可以满分

✅ 第二轮：8-9月，1小时/天
划重点：看视频，总结知识点
建立知识体系，可以画出知识图谱，没有知识框架，知识是零碎的，有了之后，知识会变得系统化，知识点之间会具有更强的关联系，有助于联想记忆。

✅ 第三轮：10-11月，1-2小时/天
划重点：看辅导书，背诵知识点
背诵知识点不能太早，也不能太晚。太早会忘记，太晚的话，与背大题冲突，因此放在11月份刚刚好，每天背诵1-2小时。

✅ 第四轮：12月，每天2-4小时
划重点：背大题，背大题，背大题
肖大爷永远是你大爷，熟读肖八，背诵肖四

英语: 目标70分

[1] 计划分数 -> 70
英语一: 70分 = 完形填空5分 + 阅读理解40分 + 翻译5分 + 作文20分
英语二: 70分 = 完形填空5分 + 阅读理解40分 + 翻译9分 + 作文16分

考试时间3小时，180分钟，推荐做题顺序：作文 -> 阅读理解 -> 翻译 -> 完形填空做题

  • 小作为15分钟
  • 大作文35分钟
  • 阅读理解90分钟，约18分钟/篇
  • 翻译20分钟
  • 完形填空20分钟

❤ 要求 : 作文较好完成要求（翻译和作文的要求在下面会详细列出）；阅读理解最多错5个左右；翻译基本完成要求；完形填空最多错10个左右。

[2] 分值组成
试卷结构:

  • 完形填空
  • 阅读理解
  • 翻译
  • 大小作文

英语一试卷题型结构：共计52道题目，满分100分。

  • 完形填空：20小题，每小题0.5分，共10分
  • 阅读理解：25小题，没小题2分，共50分
  • 翻译：5小题，每小题2分，共10分
  • 写作：2小题，小作文10分，大作文20分，共30分

英语二试卷题型结构：共计48道题目，满分100分。

  • 完形填空：20小题，每小题0.5分，共10分
  • 阅读理解：25小题，没小题2分，共50分
  • 翻译：1小题，共15分
  • 写作：2小题，小作文10分，大作文15分，共25分

真题！真题！真题！

做笔记！做笔记！做笔记！

[3] 复习规划

✅第一阶段：3-6月
划重点：单词+精读传统阅读

✅第二阶段：7-9月划重点：单词+选择搭配+翻译
选择搭配，即新题型，主要考察排序、七选五、小标题匹配。难度排序>七选五>小标题匹配英语一主要考查的题型是排序题、七选五，英语二主要考查的题型是七选五、小标题匹配。英语一和英语二的选择搭配难度系数差别比较大，英语一比较难，英语二比较简单，完全可以满分。做题前先确认新题型属于哪一类填。

✅第三阶段：10月划重点：单词+完形填空
完型填空很多同学都选择了放弃治疗，原因是开始复习节奏比较慢导致后期复习时间不足，但近几年的完形填空难度有所下降，虽说20个选择才10分，但要上70这部分也不能忽略。每天尽量做2套真题的完型，每篇完型的时间控制在20分钟之内！ 反复练习真题。

✅第四阶段：11-12月
划重点：单词+作文

最后一个小tip，练练字，作文评分原则的第5条说的就是这点。

㊙️ 额外福利：考研英语成绩70分及以上，研究生可以免修必修英语课，是不是很棒 。

数学: 目标120分

[1] 计划分数 -> 120

120分 ≈ 选题择24分 + 填空题16分 + 解答题80

考试180分钟，按试卷顺序做题即可

  • 单选题24分钟，约1.5分钟/题
  • 多选题42分钟，约2.5分钟/题
  • 分析题100分钟，约20分钟/题
  • 剩余14分钟可以查漏补缺

❤ 要求: 选择题最多错2个左右；填空题最多错2个；大题最多1.5道题不会

[2] 分值组成
试卷题型结构：共计23道题目，满分150分。

  • 单项选择题：8小题，每小题4分，共32分
  • 填空题：6小题，每小题4分，共24分
  • 解答题(包括证明题)：9小题，共94分

数学一分值分布：

  • 高等数学56%，84分
  • 线性代数22%，33分
  • 概率论与数理统计22%，33分

数学二分值分布：

  • 高等数学78%，127分
  • 线性代数22%，33分

[3] 复习规划

✅基础阶段：3-5月
划重点：记笔记！记笔记！记笔记！把老师的手抄全部记下来。
既然是基础阶段，那就明确此阶段的目标是学会大纲考的所有知识点，还没到大量练习的时候，只需要少量习题来帮助理解知识点，此阶段是需要学会运用知识做题。刚开始做题时，不要那么猛，上来就写综合性的，一个一个来。

✅强化阶段：6-7月
划重点：记笔记！记笔记！记笔记！把老师的手抄全部记下来。
计算能力很重要，千万不要觉得老师一讲，就明白的透透彻彻，【套用本山大叔一句话，没病走两步】，不动手，都不知道自己到底什么样，考试交的是试卷，所以要做到熟练计算。

✅提高阶段：8-9月
划重点：记笔记！记笔记！记笔记！把真题试卷错误题都整理出来。

✅冲刺阶段：10-12月
半年多的笔记派上用场了，自己之前总结过的知识点，例题，错误，拿出来快速过，容易忘记的公式加强记忆，错误的习题不要再犯。
推荐资料：真题+笔记+辅导书+模拟题

专业课: 目标120分

[1] 计划分数 -> 120

[2] 分值组成

注意考试大纲！注意考试大纲！注意考试大纲！

[3] 复习规划

✅基础阶段：4-5月
划重点：理解每个知识点
看课本，看视频，看资料，看课件，有什么看什么，别浪费，把不会的标记出来，复习完后回头再看一遍。

✅提高阶段：6-7月
划重点：总结
整理出自己的知识图谱，同样按费曼逻辑，虽然花费一些时间，但对于建立知识架构，串联知识点有非常大的帮助。
对于基础阶段没有学会的知识点，此阶段必须要克服。

✅强化阶段：8-9月
划重点：有真题做真题，没有真题做习题，有什么题做什么题

✅冲刺阶段：10-12月
主要是依靠前期和中期梳理总结的笔记，习题，把书籍和视频当资料查，不断重复强化知识点。
